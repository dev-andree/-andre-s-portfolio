Portifolio
Ao longo do curso, aprendemos a utilizar diversas ferramentas essenciais para o desenvolvimento web. Começamos com a linguagem de marcação HTML (HyperText Markup Language), que é a base de qualquer página da web. Com HTML, conseguimos estruturar o conteúdo, definindo elementos como títulos, parágrafos, listas, links, imagens e tabelas. É a espinha dorsal que dá forma e organização à informação apresentada nos sites.
Em seguida, mergulhamos na estilização com CSS (Cascading Style Sheets). Essa linguagem nos permite adicionar cor, tipografia, layout e outros aspectos visuais às páginas web, criando uma experiência de usuário atraente e intuitiva. Com CSS, aprendemos a posicionar elementos, criar animações e aplicar estilos responsivos para diferentes dispositivos, garantindo que as páginas sejam visualmente agradáveis e funcionais em qualquer tela.
Além disso, estudamos a linguagem de programação JavaScript, fundamental para adicionar interatividade e dinamismo às páginas web. Com JavaScript, conseguimos manipular o DOM (Document Object Model), responder a eventos do usuário, validar formulários, criar gráficos interativos, entre outras funcionalidades. Essa linguagem torna as páginas web mais envolventes e interativas, proporcionando uma experiência mais rica para os usuários.
Por fim, exploramos a linguagem de programação C, que é amplamente utilizada em diversas áreas da computação, desde sistemas operacionais até aplicações embarcadas. Aprendemos sobre conceitos fundamentais como variáveis, loops, funções, estruturas de dados e gerenciamento de memória. A linguagem C nos proporcionou uma base sólida para entender a lógica de programação e desenvolver algoritmos eficientes.
